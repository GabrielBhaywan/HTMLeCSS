codico rgb

a sigla rgb significa r = red g = green b = blue

O rgb: vai quase da mesma forma que o exdecimal QUASE,
red 0 até 255, green 0 até 255, blue 0 até 255;

O rgb não usa letras porem usa mais digitos sendo 0 o mais fraco/nada/minimo e 255 o mais forte/tudo/maximo


exmplo aqui vermelho rgb(255, 0, 0);

exmplo aqui verde rgb(0, 255, 0);

exmplo aqui azul rgb(0, 0, 255);

codigo HSL

H = hue

Representa a cor propriamente dita e é medida em graus, variando de 0 a 360. Por exemplo, 0° é vermelho, 120° é verde e 240° é azul

S =saturation

Indica a intensidade da cor, variando de 0% (um tom de cinza) a 100% (a cor mais vibrante).

L =Lightness

Refere-se à claridade da cor, variando de 0% (preto) a 100% (branco).

exmplo aqui vermelho hsl(0, 100%, 50%);

exmplo aqui verde hsl(120, 100%, 50%);

exmplo aqui azul hsl(240, 100%, 50%);
